Tutorial on JavaScript variables for Beginner

Overview about JavaScript

Javascript is a high-level programming language that is popular in creating web. Javascript helps increase interactivity in webs. It was created to use to make web more lively but now it is used in different coding environments.

How to name variables in JavaScript

Variable name must not have special case except ‘$’ and ‘_’. First character in variable name must not be number like 1,2,3. Variable name can contain number or alphabetic character. It is better to have a semantic name for variables.

Data types

JavaScript has different data types:

  • Number: integer or floating number
  • String: a sequence of text that wrapped inside a single or double quoted. Example: ‘Hello World!’ or “Hello World”

You can learn more about Javascript Data types in here.

JavaScript Syntax

Javascript Syntax is a collection of rules about types of values. There are two types of values:

  • Fixed values or literal
  • Variable values or variables (var)

Fixed Values variable values or variables (var)

Variable values are values when the data type of variable is undefined. In other words, variable is like a container and different types of value can be stored inside variable. However, with fixed values, only values with same data type can be stored.

Differences between var, let, const:

var is a keyword that is used to declare a variable. It can be used to declare same variables many times.

In below example, var 'Name' is declared twice and value of var 'Name' is assigned with the value that is assigned after.

var Name = 'Jim';

var Name = 'Peter';

alert(Name);

javascript.info says Peter

let is similar to var but can only be used once to declare a variable.

In below example, Javascript has Syntax Error because variable 'date' is declared twice. You can find more details from here.

let date;

let date;

const refers to ‘constant’. If you use constant, it is a literal and must be assigned with a fixed value. And the value of const cannot be changed (except for arrays). It cannot be redeclared.

In below example, JavaScript says 'TypeError: Assignment to constant variable' because variable 'Age' is const and has value of 25 but later is assgined with 27.

const Age = 25;

Age = 27;

Note: Javascript is case-sensitive. Variables like FName and Fname are considered to be different variables. It is same to come with function, for example ‘let’ or ‘const’, Javascript does not recognize ‘Let’ or ‘Const’.
